## Projet 3: Analyse d’un fichier CSV

Ce projet Java permet de lire un fichier CSV, d’analyser les colonnes numériques et de calculer des statistiques (moyenne, minimum, maximum). Le programme est structuré en plusieurs classes selon les principes SOLID, avec une gestion des erreurs et la génération d’un rapport texte contenant les résultats.

## Fonctionnalités

Lecture de données : Chargement et parsing automatisé de fichiers CSV.

Analyse statistique : Calcul dynamique des métriques clés (minimum, maximum, moyenne, etc.) sur les données extraites.

Architecture modulaire : Utilisation de classes abstraites pour permettre une extension facile des types d'analyses.

Génération de rapports : Exportation des résultats sous format texte pour une lecture facilitée.
