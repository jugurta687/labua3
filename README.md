## Projet 3: Analyse d’un fichier CSV :
Ce projet Java permet de lire un fichier CSV, d’analyser les colonnes numériques et de calculer des statistiques (moyenne, minimum, maximum). Le programme est structuré en plusieurs classes selon les principes SOLID, avec une gestion des erreurs et la génération d’un rapport texte contenant les résultats.

## leS TACHE A FAIRE :
- Lecture d'un fichier
- Traitement des données
- Calcul des statistiques
- Affichage des résultats
- Génération du rapport
- Gestion des erreurs
- Organisation du code
